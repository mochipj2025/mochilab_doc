# Character Rules

M.O.C.H.I. LABOで使う共有キャラクター素材の運用ルール。

## 基本方針

- キャラクターは安心感、親しみ、案内役として使う
- 医療的な効能や診断を示す役割にしない
- 不安やコンプレックスを刺激する表現に使わない
- 過度な広告、煽り、強いセールス演出に使わない
- 日本語ミニマルUIの静かな世界観を壊さない
- モバイル画面で邪魔にならないサイズと情報量にする

## キャラクター一覧

### ケツまる

明るくやさしい案内役。表情差分が多く、注意、応援、感謝、困りごとの表現に向いている。

ケツコラム部署の担当キャラクターとして扱う。

向いている用途:

- やさしい注意
- 操作完了
- 空状態
- 軽いフィードバック
- 親しみのあるSNS表現

### もちスラ

構造や理論を観察するラボ長寄りの案内役。落ち着いた分析、説明、レビュー補助に向いている。

M.O.C.H.I. LABO全体のブランドキャラクターとして扱う。

向いている用途:

- レビュー支援
- 構造化された説明
- 分析中の状態表示
- ラボらしい案内
- Promptmakerの補助キャラ

### サロンガイド

サロン、ケア、デザイン作業をつなぐ人型の案内役。利用者に近い目線で、入力や編集のサポートに向いている。

向いている用途:

- Promptmakerの案内
- サロン向けUI
- 入力補助
- やさしい確認
- 編集作業の伴走

### 研究員ペルソナ

年代別・生活文脈別の悩みを伝える図鑑素材。キャラクター単体というより、コラムや説明のためのペルソナセットとして扱う。

通常の会話、漫画、SNSでは下の名前で呼び、公式プロフィールや設定資料ではフルネームを使う。

掛け合いの基本は「もちスラ + ケツまる + まじめちゃん」の3人構成にする。まじめちゃんは女子ラボメンバーの基本リーダーとして扱う。5人全員は必要な回だけ登場させる。

向いている用途:

- 年代別のあるある表現
- 生活文脈の整理
- コラムの図解
- SNSの読みもの
- Promptmakerの世界観参考

注意:

- 診断や医学的分類のように見せない
- 体の悩みを煽る広告表現にしない
- 「あなたはこのタイプ」と断定しない

## 禁止表現

- 治る
- 改善する
- 効果保証
- 診断
- 治療
- 医学的に証明
- 必ず変わる
- 今すぐ予約しないと損
- コンプレックスを強く刺激する言い方

## 画像運用

- 原本は `shared-assets/characters/*/original/` に置く
- UI用に加工した画像は、必要に応じて `transparent/` や `ui/` を作って分ける
- 生成プロンプトやキャラ設定は `prompts/` や各キャラの `README.md` に残す
- stableに入れる前に、モバイル表示と文脈の安全性を確認する
