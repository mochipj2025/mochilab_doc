# Community Vision

M.O.C.H.I. LABOのコミュニティ構想。

## Vision

静かに試し、失敗し、気づき、直せる研究室。

人が安心して未完成のものを出せる場を目指す。

## Discord Lab

Discord研究室は、強い競争や煽りではなく、小さな実験と共有を中心にする。

## Values

- Safe Base
- 小さく試す
- 失敗を責めない
- 読みやすく伝える
- 作品を丁寧に扱う
- 強い広告感を避ける

## Possible Channels

- お知らせ
- 研究ログ
- Promptmaker実験
- UIレビュー
- キャラクター研究
- コラム案
- 雑談

## Moderation Tone

- 静か
- やさしい
- 明確
- 人を責めない
- 境界線は守る

## Guardrails

- 医療相談の場にしない
- 効果保証や診断をしない
- 過度な宣伝を避ける
- 攻撃的なレビューを避ける
- 参加者の未完成な試作を軽く扱わない
