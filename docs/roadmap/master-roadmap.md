# Master Roadmap

M.O.C.H.I. LABOは、AI × Care × Designを軸にした「思想と世界観を育てる研究室」です。

単なるコード管理ではなく、プロダクト、コンテンツ、キャラクター、コミュニティ、IPをゆっくり育てるための拠点として運用します。

## Core Concept

- 静かな日本語ミニマルUI
- モバイルファースト
- 居られるデザイン
- Safe Base思想
- Careを感じる言葉
- 過剰演出を避けた世界観
- 小さく試して改善する研究スタイル

## Phase 1: Foundation

目的: 研究室の土台を整える。

- README / AGENTS / docsの整備
- Promptmakerの安定版とdev領域の整理
- キャラクター素材の共有化
- 医療的表現、広告的表現の禁止ルール整備
- 静かな日本語UIの基準づくり

## Phase 2: Product Seeds

目的: 小さなツールを複数試す。

- Salon Promptmakerの改善
- 明るめポップ表現の選択肢追加
- ブログサイトと執筆エディターの構想
- Review Assistantの方向性検証
- Future Karte UIの実験
- Safe Base思想を反映した入力体験の設計

## Phase 3: World Building

目的: キャラクターと思想をつなげる。

- ケツまる、もちスラ、サロンガイドの役割整理
- 研究員ペルソナの図鑑化
- 漫画、コラム、SNS投稿への展開
- 世界観の言葉、色、余白、関係性の統一
- M.O.C.H.I. LABOらしい「静かな研究室」表現の確立

## Phase 4: Community and IP

目的: 継続的に育つ場へ広げる。

- Discord研究室の構想
- コミュニティ参加ルールの整理
- IP展開の可能性検討
- キャラクターグッズ、読みもの、テンプレートへの展開
- 外部公開時のブランドトーン管理

## Promptmaker

Promptmakerは、M.O.C.H.I. LABOの最初の実用プロダクトです。

目指すもの:

- 選ぶだけで使える
- 日本語が読みやすい
- サロンや小さな事業者にやさしい
- 医療的・効能保証的な表現を避ける
- 静かな世界観から、少し明るいポップまで選べる

## Blog Editor

ブログサイトと執筆エディターは、M.O.C.H.I. LABOの思想、エッセイ、コラム、キャラクター掛け合いを育てるための制作環境です。

目指すもの:

- 静かな日本語ミニマルUIで読める
- Markdownで書ける
- もちスラ、ケツまる、まじめちゃんの掛け合いを作れる
- 医療表現や過度な広告表現を避けられる
- Safe Base思想を保った読みものを増やせる

## Manga Plan

漫画計画では、キャラクターと思想をやわらかく伝えます。

- ケツまる: 親しみのある案内役
- もちスラ: 理論と観察のラボ長
- サロンガイド: 利用者に近い伴走役
- 研究員ペルソナ: 生活文脈を伝える図鑑的存在

漫画は売り込みではなく、気づきと共感を作るために使います。

## Discord Lab

Discord研究室は、静かに試し、共有し、改善するための場として構想します。

重視すること:

- 安心して居られる
- 強い競争や煽りを避ける
- 小さな実験を歓迎する
- 失敗を責めない
- 作品や考えを丁寧に扱う

## IP Expansion

IP展開は、世界観とキャラクターの一貫性を守りながら進めます。

候補:

- キャラクター素材
- SNSシリーズ
- 漫画
- テンプレート
- 小さなツール
- グッズ
- コミュニティ内コンテンツ

## Safe Base

Safe Base思想は、M.O.C.H.I. LABOの中心です。

ユーザーが迷ったり、失敗したり、うまく言語化できないときにも、戻れる場所があることを大切にします。

UI、文章、キャラクター、コミュニティのすべてで、安心して試せる状態を守ります。

## Quiet Lab

「静かな研究室」は、M.O.C.H.I. LABOの基本姿勢です。

- 余白がある
- 急かさない
- 煽らない
- 派手すぎない
- でも冷たくない
- 小さな前進を大切にする

この空気感を、すべての開発とコンテンツの基準にします。
