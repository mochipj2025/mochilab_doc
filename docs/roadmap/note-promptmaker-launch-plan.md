# Note Promptmaker Launch Plan

noteを「読む場所」から「Promptmakerを使ってみる場所」へつなげるための導線設計。

## 目的

個人サロンを運営している40代前後の人たちに向けて、投稿の負担を軽くする入口を作る。

売り込むためではなく、「投稿がめんどくさい」「何を書けばいいかわからない」という状態を受け止め、Promptmakerを無料で触ってもらう。

## 対象読者

- 40代前後の個人サロン運営者
- ひとりサロン、少人数サロン
- 施術や接客は大切にしている
- SNS投稿が負担になっている
- 売り込みっぽい発信が苦手
- 医療表現や効能表現が少し不安
- AIを使ってみたいが、何から始めればよいかわからない

## 中心メッセージ

投稿がめんどくさいのは、やる気がないからではなく、毎回ゼロから考えているから。

Promptmakerは、文章を代わりに作る道具というより、考える負担を減らす道具。

## noteの役割

- 共感を作る
- 無料開放の入口にする
- 取説へ案内する
- 小さな成功体験を作る
- セラピスト×AIの実践ログを育てる

## 最初の導線

1. note記事を読む
2. 「投稿がめんどくさい」という気持ちを受け止める
3. Promptmaker無料開放を案内する
4. 取説に進む
5. まず1投稿だけ作る
6. 作れた人向けに次の記事へ誘導する

## 最初の記事

タイトル候補:

- 投稿がめんどくさい個人サロンさんへ。Promptmakerを無料開放します
- サロン投稿、毎回ゼロから考えなくていい
- 売り込まずに伝えるためのPromptmaker取説
- 40代ひとりサロンのための、やさしいAI投稿ツール
- 投稿が止まっている日に使う、サロンPromptmaker

推奨タイトル:

投稿がめんどくさい個人サロンさんへ。Promptmakerを無料開放します

## CTA

強く売り込まない。

使うCTA:

- うまく書こうとしなくて大丈夫です
- ゼロから考えないために使ってください

- 今すぐやらないと損
- 集客できるようになります
- 売上が上がります
- 必ず反応が増えます
