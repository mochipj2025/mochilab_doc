Image2.0 character note

Character Scene Prompt Maker

表情、髪、目、衣装、動き、背景、短い詩を選んで、Image2.0向けの自然なキャラクターシーンPromptを作ります。

Photo Book Kit写真集風ページを作る Manga Scene Kit縦スクロール漫画を作る Character Card Kitキャラカードを作る Three View Kit三面図を作る LP Asset KitLP / OGP素材を作る Prompt Admin採用Promptを記録する 短文キャラPrompt日本語だけで作る

選ぶだけでPromptが育ちます。

まずはランダム生成、または気になる項目をひとつ選んでください。

プリセットはあとから自由に足したり消したりできます。

1. 表情・口元

目とは別に、顔全体の空気や口元の印象を選びます。

+

2. アクセサリー

顔や頭まわりの装備、小物を選びます。

+

3. 性格・趣味

キャラの内側や好きなものを選びます。

+

4. 世界観属性

キャラが属している世界の雰囲気を選びます。

+

5. 頭身

キャラの見た目年齢ではなく、絵のバランスだけを選びます。

+

6. 髪

前髪と髪型を分けて選べます。髪色などは自由入力で足せます。

+

7. 目

目の形と色を分けて選べます。視線やまつげは自由入力で足せます。

+

8. 衣装

服装だけをカテゴリごとに選べます。顔や性格と混ざりすぎないように出力します。

+

9. 動詞

キャラが何をしているかを選びます。

+

10. 背景

場所や空気感を選びます。

+

11. 詩

短い一文を入れると、Promptの余韻になります。

+

12. 画風

イラスト系とフォトリアル系を選べます。品質呪文は使いません。

+

フォトリアル追加オプション

フォトリアル系を選んだ時だけ、背景描写のあとに自然に追加します。

13. 出力タイプ

キャラを何として出すかを選びます。

+

14. 映す場所

正面、横、バストアップなど、キャラをどこまで映すかを選びます。

+

15. キャラJSONメモ

JSON資産として残すための名前、ID、メモを整えます。

+

生成結果
