# M.O.C.H.I. LABO

AI × Care × Design experimental lab.

日本語ミニマルUIとモバイルファーストの体験を研究・開発するための小さなAI研究室です。

## Projects

- Promptmaker
- Review Assistant
- Future Karte UI

## Philosophy

試す
↓
失敗する
↓
気づく
↓
直す

小さく作って改善するAI研究室。

## 目的

- AIを使った創作・支援・レビュー体験の設計
- Careの思想を取り入れたやさしいUI/UX
- 日本語で自然に使えるミニマルな画面設計
- 複数プロジェクトへ展開できる再利用可能な構造

## 構成

- `docs/`: ブランド、UI、思想の共通ドキュメント
- `promptmaker/`: プロンプト作成支援テンプレート
- `review-assistant/`: レビュー支援テンプレート
- `future-karte/`: 未来カルテ系の実験テンプレート
- `shared-assets/`: ロゴ、キャラクター、テクスチャなどの共通素材

## 基本方針

小さく始め、丁寧に検証し、再利用できる形で残します。
