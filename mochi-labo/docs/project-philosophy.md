# Project Philosophy

## AI × Care × Design

mochi-laboは、AIを効率化だけのために使うのではなく、人の不安を減らし、考える力や創る力を支える存在として設計します。

## Care

Careは、やさしい見た目だけではありません。

- 迷ったときに戻れる
- 失敗しても安心できる
- 状態がわかる
- 次に何をすればよいかが伝わる
- 使う人の時間と集中を大切にする

## Design

Designは、情報、操作、言葉、余白を整えることです。

派手さよりも、毎日使える静かな強さを大切にします。

## Template

テンプレートは完成品ではなく、次の制作を速く、丁寧にするための土台です。

各プロジェクトで得た知見は、必要に応じて `stable/` や `docs/` に戻します。
