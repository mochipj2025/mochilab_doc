# Promptmaker

M.O.C.H.I. LABOのプロンプト制作ツール置き場です。

## Tools

### あなただけの相棒maker

ランダムな言葉の組み合わせから、小さな相棒キャラクターの三面図プロンプトを作る試作品です。

- 場所: `dev/companion-maker.html`
- 入口: `index.html`
- できること:
  - ベース、モチーフ、色、目、チーク、口、アイテム、アクセント、性格をランダム生成
  - 生成されたキャラ要素と固定プロンプトを合体
  - 完成プロンプトをコピー

## Notes

- `dev/` は開発中の試作品です。
- `stable/` に移す前に、モバイル表示、文言、素材の扱いを確認します。
- 医療的な診断・効能表現には使いません。
