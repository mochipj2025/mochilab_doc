# Promptmaker Dev

開発中の試作品と、公開前のメモを置く場所。

## 試作品

- `companion-maker.html`
  - 小さな相棒キャラクターの三面図Promptを作る試作品
  - 本番化する場合は、相棒専用Kitとして整理する

- `character-promptmaker/`
  - Prompt化する前のキャラクター設定メモを作る試作品
  - 現在の本命ツールではなく、発想メモ用
  - トップには出さない

- `image2-character-promptmaker.html`
  - 旧入口
  - `character-promptmaker/` へ転送するだけなので、トップには出さない
