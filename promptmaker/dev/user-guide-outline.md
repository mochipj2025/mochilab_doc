# Promptmaker User Guide Outline

サロンPromptmaker無料開放時に使う取説の骨子。

## 目的

投稿がめんどくさい個人サロン運営者が、迷わず1投稿を作れるようにする。

## 対象

- 40代前後の個人サロン運営者
- SNS投稿が負担
- AIに慣れていない
- 売り込み表現が苦手
- 医療表現が不安

## 取説の基本方針

- 長く説明しすぎない
- 最初のゴールは1投稿だけ
- 画面ごとに何をすればいいかを書く
- 専門用語を避ける
- できた投稿を完璧にしようとしない

## 取説構成

### 1. Promptmakerとは

Promptmakerは、AIに渡すための文章を作る道具です。

あなたの代わりにサロンを語るものではなく、投稿を考える負担を減らすために使います。

### 2. 最初に作る投稿

おすすめは「空き枠のお知らせ」。

理由:

- 目的が明確
- 書く内容が少ない
- すぐ投稿に使いやすい

### 3. 入力すること

- 投稿の目的
- サロンの空気感
- メニューや案内内容
- 避けたい表現
- 文章のトーン

### 4. 空気感の選び方

- 静か
- 上品
- 透明感
- やわらかポップ
- フレッシュポップ
- 編集風ポップ

### 5. AIに貼り付ける

Promptmakerでできた文章を、ChatGPTなどに貼り付ける。

出てきた文章は、そのまま使わず、自分の言葉に少し整える。

### 6. 確認すること

- 強く売り込んでいないか
- 医療的な表現になっていないか
- 読みにくくないか
- 自分のお店らしいか
- 投稿する自分がしんどくないか

### 7. うまくいかないとき

- 短くしてもらう
- やさしい言い方にしてもらう
- 売り込み感を減らしてもらう
- 医療表現を避けて言い換えてもらう
- もう少し自分のお店らしくしてもらう

## 最初のテンプレート

### 空き枠のお知らせ

用途:

今週や近日の予約可能時間を、押し売りせずに案内する。

言い方:

- 今週、少しゆっくりご案内できる時間があります
- 午前中に空きが出ました
- 静かに過ごしたい方におすすめの時間です

避ける:

- 早い者勝ち
- 今すぐ予約しないと損
- 絶対おすすめ

### 季節のお知らせ

用途:

季節の変わり目に、ケアのきっかけを作る。

言い方:

- 季節の変わり目に、少し自分を整える時間を
- 忙しい日が続く前に、ひと息つく時間を

避ける:

- 改善します
- 治します
- 効果があります
