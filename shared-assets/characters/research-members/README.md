## Usage Notes

- 医療的な診断、治療、効能保証として使わない
- 悩みを煽ったり、コンプレックスを刺激する広告表現にしない

- UIに使う場合は、1画面に詰め込みすぎず、モバイルで読めるサイズにする
- stableに入れる前に文言の安全性と可読性を確認する
