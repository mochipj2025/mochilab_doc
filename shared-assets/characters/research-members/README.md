## Role

- 世代別の悩みや生活背景を伝える図鑑素材
- ケツまる、もちスラと組み合わせる説明用素材
- コラム、SNS、Promptmakerの参考世界観

## Naming Rules

- 通常の会話、漫画、SNSでは下の名前で呼ぶ
- 公式プロフィール、図鑑、設定資料ではフルネームを使う
- 通称は初期識別やネタとして残してよいが、人格として扱う場面では名前を優先する

## Brand Roles

- もちスラ: M.O.C.H.I. LABO全体のブランドキャラクター / ラボ長的存在
- ケツまる: ケツコラム部署の担当キャラクター / お尻まわりの読みもの案内役
- 5人の研究員: ケツコラム部署に登場するペルソナ兼ラボメンバー

## Dialogue Unit

基本の掛け合いは、以下の3人構成にする。

- もちスラ
- ケツまる
- まじめちゃん

5人全員を毎回出すのではなく、まじめちゃんが女子ラボメンバーの基本リーダーとして進行し、必要な回だけ他のメンバーがゲストとして登場する。

名前や口調の詳細案は `profile-ideas.md` に保存する。

## Usage Notes

- 医療的な診断、治療、効能保証として使わない
- 悩みを煽ったり、コンプレックスを刺激する広告表現にしない
- 「あるある」「生活文脈」「ケアのきっかけ」として扱う
- UIに使う場合は、1画面に詰め込みすぎず、モバイルで読めるサイズにする
- stableに入れる前に文言の安全性と可読性を確認する
