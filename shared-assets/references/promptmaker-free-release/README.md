# Promptmaker Free Release References

Promptmaker無料開放のnote記事、Instagram投稿、取説に使う図解素材。

## Assets

- `original/promptmaker-free-release-hero-wide.png`
- `original/promptmaker-free-release-problem.png`
- `original/promptmaker-free-release-howto.png`
- `original/promptmaker-free-release-wording-safety.png`
- `original/promptmaker-free-release-closing-wide.png`

## Usage

### promptmaker-free-release-hero-wide.png

用途:

- note記事の冒頭画像
- 自前サイトの導線バナー
- Instagram告知の1枚目

### promptmaker-free-release-problem.png

用途:

- 投稿が止まる理由の説明
- 「毎回ゼロから考えている」セクション
- Instagramカルーセルの問題提起

### promptmaker-free-release-howto.png

用途:

- Promptmakerの使い方説明
- 取説記事
- Instagramカルーセルの手順説明

### promptmaker-free-release-wording-safety.png

用途:

- 医療表現や効果保証を避ける説明
- サロン投稿の言い換え例
- NG表現チェック

### promptmaker-free-release-closing-wide.png

用途:

- note記事の締め画像
- 自前サイトのCTA
- Instagram告知の最終ページ

## Notes

- noteに使う場合は、本文の読みやすさを優先して入れすぎない
- Instagramではカルーセルとして順番に使いやすい
- 医療表現を避ける図解は、断定や効能保証に見えない文脈で使う
- 原本は `original/` に保存し、必要に応じて圧縮版を別フォルダに作る
