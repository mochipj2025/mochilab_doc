## memoasset2 usage

`D:\00000\Obsidian_Base\notebookLMpdf\memoasset\memoasset2` から、WebUIで使いやすい名前に整理した素材。

- `memo-mochi-idle.png`: 待機・案内用のもちスラ。必要時のみ使う。
- `memo-mochi-mic.png`: 録音入口の状態アイコン。録音パネル左上に置く。
- `memo-waveform-glow.png`: 音声入力の状態表示。録音中/待機中の視覚的なリズムとして使う。
- `memo-lantern.png`: アプリ全体の灯り。ヘッダー付近に薄く置き、主張させない。
- `memo-notebook.png`: 保存先・メモ保管の象徴。保存設定パネルの透かしに使う。
- `memo-researcher-desk.png`: 空状態の補助。本文が空のときだけ、紙面右下に小さく出す。
- `memo-saved-toast.png`: 保存完了通知の背景。保存時だけ短く表示する。

使い方の原則:

- 大判イラストを画面背景として全面表示しない。
- キャラクターは操作や状態に意味がある場所だけに置く。
- 1画面で強く見せるキャラクター素材は1つまでにする。
- 本文入力面を常に最優先にする。
